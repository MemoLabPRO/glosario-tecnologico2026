## 🚀 ¿Cómo verlo? (3 opciones — elige la más fácil)

### ✅ Opción 1 — Sin instalar nada (la más fácil)

1. Descarga o clona este repositorio
2. Abre el archivo `index.html` directamente en tu navegador (doble clic)
3. ¡Listo! No necesitas servidor, ni Node.js, ni nada.

```
📁 glosario-tecnologico/
   └── index.html   ← abre este archivo
```

---

### 🌐 Opción 2 — GitHub Pages (gratis, en línea para siempre)

Si tienes una cuenta en GitHub:

1. Sube este repositorio a GitHub
2. Ve a **Settings → Pages**
3. En *Branch*, selecciona `main` y carpeta `/root`
4. Haz clic en **Save**
5. En unos segundos tu glosario estará en: `https://TU-USUARIO.github.io/glosario-tecnologico2026`

---

### 💻 Opción 3 — Servidor local (opcional, para desarrolladores)

Si tienes Python instalado:

```bash
# Python 3
python -m http.server 8080
```

Luego abre: [http://localhost:8080](http://localhost:8080)

## 🔄 ¿Cómo contribuir o agregar términos?

Este es un documento vivo. Si escuchas un término nuevo que no está aquí:

1. Abre `index.html` en un editor de texto
2. Busca la sección correspondiente dentro del array `DATA`
3. Agrega el término siguiendo este formato:

```javascript
{ name:"Nombre del Término", tag:"SIGLA", stars:"★★★", def:"Definición clara y práctica." },
```

4. Guarda y recarga el navegador — aparece al instante.
